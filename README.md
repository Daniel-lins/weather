# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/Daniel-lins/weather
```

```bash
# Install Dependencies
$ yarn add

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
